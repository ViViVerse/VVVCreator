You have successfully created a new MOBA CAN Sensor!

A .h and a .cpp file have been created in the 'header' and 'source' directories of the collection folder. This is what you have to do now:
  1. Add the .h and .cpp files to the collection project files (MSBuild and CMake).

  2. Enter the species name into the symbiosis library header of the package (organelle/common/include/vvv_moba_symbiosis.h):

    inline const wchar species_moba_xxx[] = L"MobaXxx";

  3. Make an organelle info entry into the collection library header (organelle/collection/include/vvv_moba_library.h):

    extern const organelle_info moba_xxx_org_info;

  4. Reference the organelle info in the module of the package (organelle/module/source/vvv_moba.cpp):

    &moba_library::moba_xxx_org_info,

After this, you can build the the package.
